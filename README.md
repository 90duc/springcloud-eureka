# springcloud-eureka
本项目是搭建基于eureka注册中心的springcloud，使用zuul网关和gateway网关

<br/>
1、项目结构

eureka-server  eureka注册中心

micro-service  服务提供者

zuul-gateway  zuul网关

springcloud-gateway  gateway网关

<br/>
2、环境

eureka-server 2.1.3

springboot 2.1.9

springcloud Greenwich

<br/>
3、项目eureka单点注册注册中心地址

localhost:8180
